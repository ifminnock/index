Apply the Casagrande criterion and USCS-based frost susceptibility classification to determine if materials meet pavement structural requirements.
🔬 Grain Size Analysis Input
Casagrande Criterion (1931) A soil is frost susceptible if >3% of grains by weight are finer than 0.02 mm (20 μm) AND the capillary rise is sufficient to supply water during freezing. In uniform soils (Cu < 5), the threshold is 10% passing 0.02 mm.

⚗ Key Equations
Excess Fines — Predicted Deflection (Chapter 3) EFFi = ΔSRF × (P200 − Pcr)0.8
EFFt = Σ EFFi
If EFFt = 0: Dp = 0.034
If EFFt > 0: Dp = 0.056 + 0.0035 × EFFt
SRF Equation (Figure 3-5) SRF = −7.6477e−7·y⁴ + 9.7898e−5·y³ − 0.004624·y² + 0.10298·y − 0.034613
(y = depth in inches; if negative, set SRF = 0)
TAI Fatigue Equation (Chapter 4) Nf = C × 0.07958 × εt−3.291 × |E*|0.854
C = 10M ; M = 4.84 × (Vb/(Vv+Vb) − 0.69)
Per Ullidtz Functional Failure (Chapter 4) Nf = 3.069×109 × b × (E/E₀)0.36 × σv−2.82
R = 2.75 (Alaska regional factor); E₀ = 23,000 psi
ESAL Growth Formula ESALs = ESALday × 365 × [(1+g)n − 1] / g
g = growth rate; n = design period years
Frost Depth Estimate (Stefan equation) z = Cf × √FI
Cf = material coefficient (≈ 1.3–1.8 for typical subgrades)
